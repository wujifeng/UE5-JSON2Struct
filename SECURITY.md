# Security Policy

## Supported Versions

适用引擎版本范围：`UE 5.1.1 - 5.7.4`。  
当前以 `main` 分支中的 `Plugins/JSON2Struct` 最新代码为主要维护目标。

## Reporting a Vulnerability

请勿在公开 Issue 中直接披露安全漏洞细节。  
建议通过私密渠道联系维护者，并提供：

- 漏洞影响范围
- 复现步骤
- 可能的利用条件
- 修复建议（如有）

我们会在确认后尽快评估、修复并同步结果。

## Scope

本安全策略仅覆盖 `Plugins/JSON2Struct` 插件代码，不覆盖宿主项目其他模块。
